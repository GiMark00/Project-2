Project-2.

Этот проект - сайт с информацией об интересных местах России. Он был сделан в процессе обучения с целью закрепить навыки адаптивной вёрстки.
 
Чтобы посмотреть сайт нужно скачать файл GiMark_Russia_project4.rar, распаковать его и запустить в любом браузере файл index.html. Весь код сайта находится в нём, а также в папках blocks и pages.
